# üìù Simple To-Do List (Python)

A **command-line to-do list application** written in Python.  
It allows users to add, view, and delete tasks, with **data persistence** using a JSON file.

This project was created for **learning purposes**, focusing on core Python fundamentals and good practices.

---

## Features

- Add new tasks with validation
- Display all tasks
- Delete tasks by index
- Automatic saving to a `tasks.json` file
- Tasks persist between program executions

---

##  Purpose
Learn:
- File handling in Python (`open`, read, write)
- Using `json` for data persistence
- Basic usage of the `os` module
- User input validation
- Organizing code with functions
- Building an interactive CLI program

---

## Technologies
- Python 3
- json module
- os module
