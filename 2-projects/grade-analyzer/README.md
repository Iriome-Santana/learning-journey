# Grade Statistics Analyzer (Python)

Python utility script that analyzes a list of grades and returns basic statistics in a clean and reusable way.

## Features
- Calculates the average grade
- Finds the highest and lowest grades
- Counts passing and failing grades
- Handles empty input safely
- Uses structured return data (dictionary)


## Technologies
- Python 3

## Purpose
This project was created to practice:
- Function design
- Returning structured data
- Clean separation between logic and output
- Writing maintainable Python code
