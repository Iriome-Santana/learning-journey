# Python Fundamentals

This folder contains practice scripts focused on learning and reinforcing core Python concepts.

## Topics Covered
- Variables and data types
- Conditionals
- Loops
- Functions
- Basic data structures
- Error handling

Each subfolder contains small scripts and exercises created while learning and practicing these concepts.

## Purpose
This folder is intended for:
- Practicing Python fundamentals
- Experimenting with language features
- Keeping learning exercises organized

More complete and polished projects can be found in the `projects` folder.
